## Run on

### Browser

```js
<script src="lib/AAA.js"></script>
<script>
console.log( AAA() );
</script>
```

### WebWorkers

```js
importScripts("lib/AAA.js");

console.log( AAA() );
```

### Node.js

```js
require("lib/AAA.js");

console.log( AAA() );
```

### node-webkit

